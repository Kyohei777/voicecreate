# Voice Clone Studio Pro

Qwen3-TTSを使用した音声クローンアプリケーション。

## 機能

- **Quick Clone**: 短い音声からの即座のボイスクローン
- **Deep Learn**: 長時間動画からプロファイル作成、高品質なイントネーション再現
- **Smart Selection**: プロソディ特徴量に基づくセグメント選択
- **Sentence-level Generation**: 文単位で最適なセグメントを選択して生成

## セットアップ

```bash
docker compose up -d
```

ブラウザで http://localhost:7860 にアクセス

## 必要環境

- Docker
- NVIDIA GPU (CUDA対応)
- 16GB+ VRAM推奨
